# ✅ To-Do List

Простое приложение для управления списком задач с графическим интерфейсом на PyQt6.

## 📸 Возможности

- ➕ Добавление задач с выбором приоритета (Высокий / Обычный)
- ✅ Отметка задач выполненными (чекбокс)
- ✏️ Редактирование задачи по двойному клику
- 🗑️ Удаление выбранных задач
- 🧹 Очистка выполненных задач
- 🔘 Отметить все / Снять все отметки
- 📊 Счётчик задач (всего / выполнено)
- 💾 Автосохранение в `tasks.txt`
- 📄 Экспорт/Импорт в JSON (`tasks.json`)

## 🚀 Установка и запуск

### 1. Клонировать репозиторий (или скачать файлы)
```bash
git clone https://github.com/adamkhaydaev/mini-notebook.git
cd mini-notebook
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Запустить приложение
```bash
python main.py
```

## 🛠️ Технологии

- Python 3
- PyQt6

## 📂 Файлы данных

- `tasks.txt` — автосохранение списка задач
- `tasks.json` — экспорт/импорт задач
